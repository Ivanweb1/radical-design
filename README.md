# Radical Design — редизайн сайта

Вёрстка редизайна сайта архитектурно-строительного бюро [Radical Design](https://radicaldesign.ru) по макету из Figma. Чистый HTML / CSS / JS без сборки и зависимостей — готово под натяжку на WordPress-тему.

## Страницы

| Файл | Описание |
|------|----------|
| [`index.html`](index.html) | Страница бюро (О нас): hero, проекты, клиенты, награды и СМИ, принципы работы, форма, команда, футер |
| [`project.html`](project.html) | Страница проекта («Бутик отель»): интро, слайдер, медиа, видео, авторы |

## Структура

```
├── index.html
├── project.html
└── assets/
    ├── css/style.css     — все стили
    ├── js/main.js        — курсор, бургер-меню, маркиза клиентов,
    │                        список наград, слайдер и flip-карточки принципов
    └── img/              — изображения, фигуры, иконки
```

## Особенности

- **Адаптив** — десктоп + мобильная версия по отдельным макетам (брейкпоинты 900 / 600 px)
- **Кастомный курсор** (синяя точка следует за мышью на десктопе)
- **Бургер-меню** на мобильных
- **Бегущая строка** клиентов, **flip-карточки** принципов работы (переворот по наведению)
- **Награды** — раскрытие полного списка по «Смотреть ещё»
- **Art-directed изображения** через `<picture>` (вертикальные фото на мобильном)

## Локальный запуск

Любой статический сервер, например:

```bash
npx serve .
```

Затем открыть выведенный адрес (обычно http://localhost:3000).
